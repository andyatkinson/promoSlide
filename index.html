Promo Slide

Because we need more annoying things like this on the internet! :)

This type of slide-out promotional banner is being used on lots of websites. I wanted to make a simple plugin version of this pattern so that it could be used somewhere else.

This box is 1000 pixels which should exceed your browser viewport and force scrollbars to appear. Scroll down to see a slideout promotion. Scroll up to see it disappear.
