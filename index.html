Hi! Thanks for checking out Promo Slide.

  • Promote additional pages on your site. More page views!
  • Promote a newsletter sign-up link or form in an unobtrusive way.

Email me with ideas or feedback. This demo page was made with the flatly bootstrap theme.

This plugin source is on github.

Using Promo Slide on your site

A simple example of Promo Slide is below. It customizes the header text, and puts a link in the body text.

$('body').promoSlide({
      headerText: "Read more from my blog",
      bodyText: "Latest article: <a href='#'>10 Blogging Tips</a>"
});

What does it look like?

Promo Slide uses a panel component from Bootstrap as the main container HTML. This way the panel markup is well-known, there are some built-in style variations, and additional styles can be added easily. The panel is wrapped in a plugin class promo-slide-container.

The examples below are bootstrap panels using the panel-primary, panel-info, and panel-danger CSS classes.

Known issues:

  • Not tested on browsers other than Chrome and Safari. Would love to get help here.
  • Not supporting multiple lines of body text.
  • Bootstrap panel classes have to added to plugin source now. Can make this a config option later.
